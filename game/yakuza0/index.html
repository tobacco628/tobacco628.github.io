如龙：狭间

2025年07月13日

でも　そんな俺の汚い手を握り笑ってくれた

BGM: We All Lie - Hajin

昨天看到一篇帖子，说“世俗的味道”是《如龙》系列的醍醐味：“……虽然老如龙因为技术问题会有进店黑屏转换场景，但是制作者非常用心。在黑屏的时候，如果玩家进入的是便利店，会有进入711，罗森那种叮叮叮的电子铃声。进入寿司店，则是拉动和式拉门的拖动声。进入餐厅后。如果吃的是汉堡，会有悉悉索索地打开汉堡软纸包装的声音。如果是吃牛肉饭，则会有碗筷相互碰撞的叮咚声。如果是去吃‘韩来烤肉’，则会在菜端上来的时候，发出肉块在铁板上烤制时的吱吱声。”（大意）

这一下子提醒了我。我想起曾经看过的三场电影。或许这才是我喜欢这个系列的理由。

第一场电影，发生在2019年底，我还在国内读本科。头两年学生们都被发配到郊区大学城，附近有一家涮羊肉，一家比格披萨，一些小烧烤，走一公里到地铁站能吃到黄米凉糕；但依旧荒凉得很，天一黑走在路上可能遇到暴露狂，一些墙角远远飘来尿骚味。每周末我拎着老爹洗好的衣服，挤了十几站地铁，穿过长长的廊道走出天桥，扑面而来的热浪和沙土都让我觉得好像穿越了一样。

有一天，社团的朋友问我第二天78节有没有课，她叫了另个朋友一起去附近的购物中心看《误杀》。她自来熟，发量让人嫉妒，时常语出惊人，也是她让我终于得知同学的死讯。56节是水课，课后，我背着书包和她们在校门口集合。

电影讲了什么，已经不太记得了。有几个镜头很吓人，有几处我忍不住出声咒骂。我印象深刻的是那家影院特别冷，混纺毛衣和羽绒服裹在身上跟没穿一样。寒气一阵阵地从脚心向上蔓延，马上到期末了，我一边尽力抱住双臂，一边祈祷第二天不要感冒。《We All Lie》响起时，我们随着人群走出了影院。刷卡过了闸机，阴沉的天空慢慢飘起雪花。

我当时为自己制定了很多计划。我报名了第二年暑假的交换夏令营，寒假要为某大赛当志愿者；最后一科考统计学，之后就和朋友们去重庆玩，忙得不可开交。我的统计学分数没有想象中高，让我感到有些屈辱；不过我没有感冒。

此时我在洗碗机的轰鸣声中，点开了《We All Lie (Slow Version)》。天气预报聒噪了一天的雷阵雨终于淅淅沥沥地落下，在窗户上留下和菜板一样的划痕。

Hajin喃喃自语：Is this really true? We all lie, tell you the truth.

而我想起的是放映厅散发的逼人寒气，烈日下购物袋缠在手上的触感，还有更远一点，和家人一同分享的黄米凉糕。

第二场电影，是一年后的《掬水月在手》。我有个高中同学那几年倒电影票，我们看电影全去问他。这片子我想看很久了，但约好的朋友B发现自己不得不花点时间应付考试。我们高中同班有七八个同学上了同一所学校，我那天刚好去她的学校拜访另一个朋友W。我和W吃了一碗红豆沙，给B发信息说好吧，反正票买在你们学校附近，我自己去。

记忆突然出现了断裂。聊天记录显示那天是11月初，可是我记得出门的时候自己穿了短袖。就当是夏天吧。我问了一圈，朋友都不在附近，于是步行到麦当劳吃热辣无骨鸡。豆沙很甜，鸡肉很油，花了不少时间处理个人卫生。电影买到晚上大概8点，下楼的时候天已经黑了，手机即将没电。我站在一条很宽的马路前焦急地等红绿灯，和四面八方的人擦肩而过，风都是黏的。影厅老旧，检票之后呼哧带喘地小跑进影厅，刚好开演，竟然只有我一个人。

买票之前我就知道这部纪录片很小众，还问黄牛同学什么时候它会下架。一个人看电影的感觉挺奇妙。除了自己没有旁人，大可身体放松；然而这样一来电影仅为自己播放，又或者英雄不问出处，每排座椅都是观众，压力山大。后来看豆瓣，才回过味来，它实在不算一部很好的电影；但那时看到黑幕之后月亮幽幽，乌鸦啼叫，只觉凄清。叶嘉莹独自在叶赫水边漫步，天幕灰蓝低垂，满地焦土，又让我生出些感同身受。

看完电影，工作人员也要下班了。我赶在清洁工入场之前溜走，不知为何他们的前门做得像后门，招牌和诸多广告涂鸦混在一起，不易辨认。我走上楼梯，天气不热了，甚至有些凉爽。门口是几袋垃圾，我发觉自己身处小巷。在昏黄的路灯下，我猛地想到了“空白的一坪”，想到真岛吾朗被烟头映得忽明忽暗的侧脸。

第三场电影，是两年前的《首尔之春》。这又是一个冬天，学生时代最后的寒假，我决定尽情玩乐。从暑假开始，我花了更多时间去影院，填补贫瘠得近乎艺术洼地的大脑。我办了一家影厅的会员，他们的设施不如另一家好，离得更远，但他们会放映一些老片子，那么这就是理由。我看了《幽灵公主》（居然收了会员费还要钱，我呸），《虎胆龙威》，以及一些小众得没有豆瓣评分的片子。为了回本，我有一周连着每天都去看电影，其中就包括《首尔之春》。

豆瓣给予它很高的评价，而放映它的影厅离得很远，地铁倒公交，一个多小时，下车还要步行，我小心地走在积雪旁边，毫无悬念地迟到了。影厅建在车库上，没有电梯，气喘吁吁地爬了几层摔倒能撞断肋骨的混凝土楼梯，缩头走进影厅的时候，我吓了一跳：所有座位都满了。借着微光，我看到许多戴着眼镜和毛线帽的老人，面目肃然。

我进场时正是一场厕所戏，错过了开头，所以前半程都会错意，把全斗光当成好人。士兵在阴暗之下集会，夫人默默端上饭食，不知怎的让我期待起《小姐》中章鱼逼近公爵一般的画面。但这并不是爽片，从我意识到不对劲开始，一切更加沉重，让人闻到了军大衣和皮鞋所散发的腐朽，忧郁的气味。

这部电影留给我的，正是向着某个目标奔去，进退维谷的茫然；还有尘埃落定的坚硬楼梯扶手，还有毛线帽，圆眼镜，手织围巾；还有该发生的必将发生，已发生的也必将终结，存在的终将远去，远去的也会归来。

——说回《如龙》。

《如龙》系列我目前只玩了0，那还是好几年前的杰作了。看《误杀》和《掬水月在手》时，我还是个不在意或者说根本不知道成就系统这回事的年轻人，玩游戏最重要的就是通关主线。那时我也从不用社区，写评价还要开加速器，非常麻烦。不过，我清晰地记得自己点开全部是现实好友的游戏列表，看到他们玩了某游戏的无法名状的开心，以及很少交谈游戏的寂寞。这寂寞成为一粒种子，让我一路来到如今。

第一次玩《如龙0》，就发生在《掬水月在手》那一年。断断续续地开始，奋起直追地结束，到了《首尔之春》，我已经成为可悲的成就党，开始二周目。等我恋爱，主线剧情又通了一遍。又是夏天，女友去欧洲开会，有时晚上聚会喝多了说胡话，有时买包口味独特的薯片回酒店，总是睡不着。我在油管上找睡眠音乐，将听筒靠近扬声器，等到那头传来均匀的呼吸声，我就将自己静音，连上手柄，用师兄的旧电脑点开熟悉的图标，桐生一马的扑克脸。做支线，找电话卡，一直到夜里两点多才尽兴。

女友上班和睡觉时，我都在玩如龙。边看攻略边玩游戏，多少有点影响体验，特别是视频攻略。但像《如龙》这样剧情线性，支线更加无厘头的游戏，文字攻略已经足够。我玩了八十多个小时，到桐生开始在房地产业叱咤风云，我有些累了，诸多小游戏成就搁置下来。

像0这样后出的前传，几乎一定带有某种悲剧意味。相识于微时，又相忘于江湖，这几乎是世间最常见而不忍卒读的结局。几年后再玩，锦山在山中流下的泪水，和甲板上的劝告都让我有了新的感受。我买了1，买了7，但天下一番最初的故事还没有完，我还没有将这一切画上句点。

那纠缠不清，晦暗不明的少年时代。

二周目，我开始注意此前没有注意到的细节。故事的开始，是初出茅庐的桐生一马低头盯着躺在地上的债主。天光一线，他穿着黑色的西装，佩戴风间组的铭牌。之后锦山彰打来电话，清晨6：50，他们在天极轩吃了一碗拉面，听到了电视播放的新闻。

此后我让桐生几次路过粉红大街，我都会想象那碗面是怎样冒出蒸腾的水汽，让劳碌的身体得以安歇。一夜没睡，打了架，唱了K，吃了面，和最好的朋友在一起。抛去那些暴力部分，二十岁的桐生一马的生命里，应该也有许多值得怀念的东西吧？

我也是哟。尽管很多都不是那样珍贵的记忆，却也无法轻易抛弃。

去年和女友二刷《428：被封锁的涩谷》（再不psp三刷，ADV都要出啦！），注意到一处有趣的词语释义：

“狭间：读作はざま。事物与事物之间的狭小场所。我们平时于缝隙中生存，在狭间动荡。

“理想与现实的狭间。善与恶的狭间。希望与绝望的狭间。理性与本能的狭间。爱与恨的狭间。真实与虚伪的狭间。过去与未来的狭间。光与暗的狭间。秩序与混沌的狭间。自己与他人的狭间。梦想与现实的狭间。生与死的狭间。偶然与必然的狭间。”

Is this really true? Is this really true? Is this really true?

脚下的土地与地上的身影是桐生一马命运的交叉点。蒸腾的水汽，滋滋的铁板，光怪陆离的堂吉诃德，你记得也好，最好你忘掉。向着某条道路前进时，总有一些东西潜滋暗长，你或它们，永远地走下去，一条线或一个圆，永不相交，永不消逝。
